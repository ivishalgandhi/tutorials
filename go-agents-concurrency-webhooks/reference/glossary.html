Glossary

Ubiquitous language for the course. All lessons, exercises, and quizzes use these terms consistently.

Term Definition
Agent loop The repeated sense → decide → act cycle that drives an agent until it reaches a terminal condition.
Goroutine An independently scheduled function in Go, started with the go keyword. Cheaper than an OS thread.
Channel A typed conduit for values between goroutines. Sends and receives use the <- operator.
Buffered channel A channel with capacity make(chan T, n). Sends block only when the buffer is full.
Worker pool A fixed set of long-lived goroutines that read jobs from a channel and process them concurrently.
Semaphore A concurrency limiter implemented as a buffered channel of empty structs: make(chan struct{}, n).
context.Context A request-scoped object that carries deadlines, cancellation signals, and values across API boundaries.
Webhook An HTTP callback pushed by a provider when an event occurs, rather than the consumer polling for it.
HMAC signature A hash-based message authentication code used to prove that a webhook payload came from the provider.
Idempotency key A unique identifier for a webhook event that lets the receiver process retries as no-ops.
Async acknowledgement Responding HTTP 200 immediately while processing the event in a background goroutine or queue.
Backpressure A mechanism that slows the sender when the receiver is overloaded, preventing unbounded memory growth.
Graceful shutdown Stopping new work, draining in-flight work, and then exiting cleanly on SIGTERM.
Thundering herd A sudden burst of concurrent work that saturates downstream resources.
