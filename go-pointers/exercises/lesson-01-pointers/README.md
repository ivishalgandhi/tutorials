# Lesson 1 Exercises: Pointers in Go

## Setup

Make sure you have Go installed:

```bash
go version
```

## Exercises

1. **01-fix-the-swap** — pass pointers to a function that must modify its caller's variables.
2. **02-nil-safe-receiver** — write a pointer-receiver method that handles a nil receiver gracefully.

## How to work

```bash
cd 01-fix-the-swap
go run problem.go      # see the bug
go run solution.go     # see the fix
```

Try to fix `problem.go` before looking at `solution.go`.
