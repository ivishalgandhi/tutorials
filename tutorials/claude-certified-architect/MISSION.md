# Mission: Claude Certified Architect, Foundations

## Why
I am a Director for database reliability who has been vibe-coding with Claude but needs to demonstrate
rigorous, production-grade AI architecture skills to advance my career. Passing the Claude Certified
Architect, Foundations exam gives me the first official Anthropic credential — a credible signal to
hiring managers and colleagues that I can design and ship enterprise-scale Claude systems, not just
prompt them.

## Success looks like
- I can pass the CCA exam (720/1000 scaled, 60 questions) on first attempt
- I can explain the agentic loop using `stop_reason`, not natural language parsing
- I can design a multi-agent system (orchestrator + subagents) with correct context isolation
- I can configure an MCP server, choose the right transport, and handle tool errors properly
- I can write a CLAUDE.md that anchors production behavior — distinct from personal preferences
- I can choose between plan mode and direct execution, and use hooks for deterministic guarantees
- I can apply prompt caching and context management to control cost and reliability
- I can speak the vocabulary fluently in a Director-level design review

- Prefers deep understanding over rote memorization — wants to reason from first principles
- All diagrams must be Mermaid (renderable in lessons and notes)
- Materials being built on M4 Mac at `/Users/vishal/learn/tutorials/claude-certified-architect/`
