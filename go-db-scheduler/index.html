Concurrent Database Job Scheduler in Go

Build a distributed, policy-driven maintenance scheduler that runs backup, integrity check, vacuum, and index operations across tens of thousands of SQL Server and PostgreSQL databases — completely asynchronously, at scale. Inspired by how Hermes and modern AI agent platforms implement cron-style job scheduling.

500–1,000 servers 10K–50K databases 1M+ jobs/day 50–200 concurrent jobs
