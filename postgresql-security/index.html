PostgreSQL Security — End to End

Master PostgreSQL security from a SQL Server DBA's perspective. Every lesson maps PostgreSQL concepts to their SQL Server equivalents and flags where the two diverge. Work through the lessons in order — authentication first, then roles, then hardening.

  • 01 pg_hba.conf — Authentication Configuration
    Rule structure · auth methods · first-match semantics · reload without restart · ~25 min
  • 02 LDAP Authentication & Active Directory Integration
    Simple bind vs search+bind · ldapbinddn · sAMAccountName · AD integration · ~20 min
  • 03 postgresql.conf Security Parameters
    SSL GUCs · listen_addresses · password_encryption · log parameters · ~20 min
  • 04 Roles and Privileges
    LOGIN vs NOLOGIN · GRANT/REVOKE · ALTER DEFAULT PRIVILEGES · public schema fix · ~30 min
  • 05 Custom Roles and Role Composition
    Role hierarchy · INHERIT / NOINHERIT · SET ROLE · auditing membership · ~25 min
  • 06 CI/CD Service Account Patterns
    migration_role · app_role · readonly_role · password rotation · REVOKE CONNECT · ~25 min
  • 07 Row-Level Security
    CREATE POLICY · USING vs WITH CHECK · BYPASSRLS · PERMISSIVE vs RESTRICTIVE · ~25 min
  • 08 Schema Security
    search_path hijacking · SECURITY DEFINER risks · pg_temp attack · hardening · ~25 min
  • 09 SSL/TLS — Encrypting Data in Transit
    hostssl · sslmode · mTLS · pg_stat_ssl · client certificates · ~20 min
  • 10 Auditing and Logging — pgaudit
    Session vs object audit · pgaudit.log · log parameters · SQL Server Audit mapping · ~25 min
  • 11 The Hardening Checklist
    7 categories · verification queries · 5 things SQL Server DBAs always miss · ~30 min
